Bézier 曲线通常只讨论参数 $t \in [0,1]$：$t=0$ 对应起点，$t=1$ 对应终点。但在动画预测、路径延伸和调试几何算法时，我们经常还会问一个问题：**如果让 $t$ 小于 0 或大于 1，曲线应该怎样继续？**

这个问题没有唯一答案，因为“外插”本身就是在选择区间外的行为。本文比较两种常见策略，并用一个自适应视口演示它们的差异。

## 三次 Bézier 曲线

给定四个控制点 $P_0,P_1,P_2,P_3$，三次 Bézier 曲线为：

$$
B(t)=(1-t)^3P_0+3(1-t)^2tP_1+3(1-t)t^2P_2+t^3P_3
$$

在 $[0,1]$ 内，它从 $P_0$ 出发，最终到达 $P_3$。也可以用 De Casteljau 算法，通过三层线性插值得到同一个点。

```javascript
function lerp(a, b, t) {
  return {
    x: a.x + (b.x - a.x) * t,
    y: a.y + (b.y - a.y) * t
  };
}

function bezierAt(points, t) {
  const q0 = lerp(points[0], points[1], t);
  const q1 = lerp(points[1], points[2], t);
  const q2 = lerp(points[2], points[3], t);
  const r0 = lerp(q0, q1, t);
  const r1 = lerp(q1, q2, t);
  return lerp(r0, r1, t);
}
```

## 两种外插策略

### 1. 端点切线外插

三次 Bézier 曲线在两个端点的导数是：

$$B'(0)=3(P_1-P_0), \qquad B'(1)=3(P_3-P_2)$$

因此可以在区间外沿端点切线做线性延伸：

$$
E(t)=
\begin{cases}
P_0+tB'(0), & t<0 \\
B(t), & 0\le t\le 1 \\
P_3+(t-1)B'(1), & t>1
\end{cases}
$$

这种方法保持了端点处的位置与一阶方向连续，轨迹简单、稳定，也比较容易预测。不过它只保留离开端点那一刻的运动方向，不再受另外两个控制点影响。

### 2. 三次多项式延拓

另一种做法更直接：不修改 Bézier 公式，把任意实数 $t$ 继续代入。De Casteljau 的线性插值在 $t<0$ 或 $t>1$ 时会变成线性外插，因此算法本身仍然可以计算。

这种策略延续的是原来的三次多项式，四个控制点会继续共同影响轨迹。它通常会弯曲，而且随着 $|t|$ 增大可能快速远离控制点。

> 演示中的“多项式延拓”有时也被称为自然外插，但它不是自然三次样条中的“自然边界条件”。准确地说，它只是将同一个 Bézier 多项式定义域扩展到实数。

## 如何观察演示

打开配套演示后，可以按下面的顺序比较：

1. 点击“播放”，观察红色参数点从 $t=-1$ 运动到 $t=2$；
2. 在 $t=-0.5$ 或 $t=1.5$ 处切换两种算法，比较橙色外插轨迹；
3. 拖动任意控制点，观察端点切线和多项式轨迹如何分别响应；
4. 留意右侧的视口范围，它会随着轨迹扩张自动调整。

两种策略没有绝对的优劣。若目标是沿端点当前速度继续前进，切线外插更合适；若目标是保留原曲线的完整三次函数关系，则应选择多项式延拓。

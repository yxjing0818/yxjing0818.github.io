De Casteljau 算法是计算贝塞尔（Bézier）曲线最经典、最直观的几何方法。相比于直接代入 Bernstein 多项式公式进行代数计算，De Casteljau 算法的核心思想非常简单且具有几何美感：**反复进行线性插值**。

这种方法不仅在数值上更加稳定，而且非常易于用代码实现。

### 核心思想

给定一个参数 $t$（通常在 0 到 1 之间），我们在相邻的控制点之间按照比例 $t$ 找到一个新点。把这些新得到的一组点作为下一级的“控制点”，继续重复这个按比例 $t$ 寻找新点的过程。每次操作都会让点的数量减少 1，直到最后只剩下一个点——这个点就是曲线上对应参数 $t$ 的点。

### 以三阶（Cubic）Bezier 曲线为例

三阶贝塞尔曲线需要 4 个初始控制点，我们记为 $P_0, P_1, P_2, P_3$。假设给定了一个特定的 $t$ 值：

1. **第一层插值（4 个点变 3 个点）：**
    
    - 在 $P_0$ 和 $P_1$ 连成的线段上，按比例 $t$ 取一点，记为 $q_0$。
        
    - 在 $P_1$ 和 $P_2$ 连成的线段上，按比例 $t$ 取一点，记为 $q_1$。
        
    - 在 $P_2$ 和 $P_3$ 连成的线段上，按比例 $t$ 取一点，记为 $q_2$。
        
2. **第二层插值（3 个点变 2 个点）：**
    
    - 现在把 $q_0, q_1, q_2$ 当作新的控制点。
        
    - 在 $q_0$ 和 $q_1$ 之间按比例 $t$ 取一点，记为 $r_0$。
        
    - 在 $q_1$ 和 $q_2$ 之间按比例 $t$ 取一点，记为 $r_1$。
        
3. **第三层插值（2 个点变 1 个最终点）：**
    
    - 最后，在 $r_0$ 和 $r_1$ 之间按比例 $t$ 取最终的一个点。这个点就是曲线上对应参数 $t$ 的位置。
        


C++

```
// 第一层插值
Point3D q0 = P0 * inv_u + P1 * u;
Point3D q1 = P1 * inv_u + P2 * u;
Point3D q2 = P2 * inv_u + P3 * u;

// 第二层插值
Point3D r0 = q0 * inv_u + q1 * u;
Point3D r1 = q1 * inv_u + q2 * u;

// 第三层插值 (最终点)
return r0 * inv_u + r1 * u;
```

为了更直观地感受这个“层层递进”的几何过程，取测试点：(0,0), (1,4), (4,4), (5,0)，有如下交互式的动态图解。你可以拖动滑动条来改变 $t$ 的值，观察中间的“辅助线”是如何运作并最终描绘出曲线的。

### 数学原理
证明 De Casteljau 算法与 Bezier 曲线的显式公式（基于 Bernstein 多项式）等价，最严谨且标准的方法是使用**数学归纳法**。

我们的目标是证明：通过反复线性插值（De Casteljau 算法）计算出的最终点，与直接代入 Bernstein 多项式求和公式得到的结果完全一致。

### 1. 定义与前置知识

**显式公式 (Bernstein 形式)：**

对于 $n$ 阶 Bezier 曲线，给定 $n+1$ 个控制点 $P_0, P_1, \dots, P_n$，参数 $t \in [0, 1]$ 下的曲线点定义为：

$$C(t) = \sum_{i=0}^n P_i B_{i,n}(t)$$

其中 $B_{i,n}(t)$ 是第 $n$ 阶的第 $i$ 个 Bernstein 基函数：

$$B_{i,n}(t) = \binom{n}{i} t^i (1-t)^{n-i}$$

**De Casteljau 递推关系：**

设初始控制点为零阶点，即 $P_i^0 = P_i$（其中 $i = 0, 1, \dots, n$）。

递推求第 $k$ 层的第 $i$ 个点时，进行线性插值：

$$P_i^k(t) = (1-t)P_i^{k-1}(t) + tP_{i+1}^{k-1}(t)$$

最终，算法经过 $n$ 层计算后得到的点 $P_0^n(t)$ 就是曲线上的点。

**Bernstein 多项式的递推性质（核心引理）：**

在证明中我们需要用到 Bernstein 多项式的一个基本性质，它可以由组合数的帕斯卡递推关系推导出来：

$$B_{i,k}(t) = (1-t)B_{i,k-1}(t) + tB_{i-1,k-1}(t)$$

_(注意：为了使边界条件成立，当 $i < 0$ 或 $i > k$ 时，规定 $B_{i,k}(t) = 0$)_。

### 2. 数学归纳法证明

我们需要证明一个更一般的结论：对于任意层级 $k$（$0 \le k \le n$）和任意有效索引 $i$，第 $k$ 层的点可以表示为初始控制点的 Bernstein 组合：

$$P_i^k(t) = \sum_{j=0}^k P_{i+j} B_{j,k}(t)$$

**基础情况 (Base Case)：**

当 $k = 0$ 时，根据定义，只需计算多项式的第 $0$ 项：

$$P_i^0(t) = \sum_{j=0}^0 P_{i+j} B_{j,0}(t) = P_i B_{0,0}(t)$$

由于 $B_{0,0}(t) = \binom{0}{0} t^0 (1-t)^0 = 1$，因此：

$$P_i^0(t) = P_i$$

这与 De Casteljau 算法的初始条件完全一致，基础情况成立。

**归纳假设 (Inductive Hypothesis)：**

假设对于层级 $k-1$，公式成立，即：

$$P_i^{k-1}(t) = \sum_{j=0}^{k-1} P_{i+j} B_{j,k-1}(t)$$

同时对于 $i+1$ 也有：

$$P_{i+1}^{k-1}(t) = \sum_{j=0}^{k-1} P_{i+1+j} B_{j,k-1}(t)$$

**归纳步骤 (Inductive Step)：**

现在我们推导层级 $k$ 的表达式 $P_i^k(t)$。根据 De Casteljau 算法的定义：

$$P_i^k(t) = (1-t)P_i^{k-1}(t) + tP_{i+1}^{k-1}(t)$$

将归纳假设代入：

$$P_i^k(t) = (1-t) \sum_{j=0}^{k-1} P_{i+j} B_{j,k-1}(t) + t \sum_{j=0}^{k-1} P_{i+1+j} B_{j,k-1}(t)$$

为了合并这两个求和式，我们将第二个求和式的索引平移（令 $m = j+1$，则 $j = m-1$。当 $j$ 从 $0$ 变到 $k-1$ 时，$m$ 从 $1$ 变到 $k$。为了统一符号，我们依然用 $j$ 表示索引）：

$$P_i^k(t) = \sum_{j=0}^{k-1} P_{i+j} (1-t) B_{j,k-1}(t) + \sum_{j=1}^k P_{i+j} t B_{j-1,k-1}(t)$$

由于规定超出范围的 Bernstein 基函数为 $0$（即 $B_{-1,k-1}(t) = 0$ 且 $B_{k,k-1}(t) = 0$），我们可以将两个求和式的范围都安全地扩展到 $0$ 至 $k$：

$$P_i^k(t) = \sum_{j=0}^k P_{i+j} (1-t) B_{j,k-1}(t) + \sum_{j=0}^k P_{i+j} t B_{j-1,k-1}(t)$$

提取公因式 $P_{i+j}$：

$$P_i^k(t) = \sum_{j=0}^k P_{i+j} \left[ (1-t) B_{j,k-1}(t) + t B_{j-1,k-1}(t) \right]$$

根据前置知识中 Bernstein 多项式的递推性质，中括号内的部分正好等于 $B_{j,k}(t)$。因此：

$$P_i^k(t) = \sum_{j=0}^k P_{i+j} B_{j,k}(t)$$

归纳步骤成立。

### 3. 结论

通过数学归纳法证明了上述一般公式后，我们只需令 $k = n$ 且 $i = 0$。这就代表了经过 $n$ 次完整迭代后计算出的最终点：

$$P_0^n(t) = \sum_{j=0}^n P_j B_{j,n}(t)$$

这个结果正好等价于 $n$ 阶 Bezier 曲线的显式代数定义 $C(t)$。因此，De Casteljau 算法这种通过多层线性插值的几何操作，在数学上完全等效于使用 Bernstein 多项式进行的求和计算。
